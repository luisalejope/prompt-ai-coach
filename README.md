Convierte ChatGPT o Gemini en un **entrenador de fuerza basado en ciencia** que crea rutinas personalizadas, claras y efectivas.

## 🚀 ¿Qué es esto?

Este repositorio contiene un **prompt optimizado** que transforma cualquier modelo de lenguaje en un:

- 🏋️ Coach de fuerza basado en evidencia  
- 📊 Generador de rutinas estructuradas  
- 🎯 Sistema guiado por objetivos reales (fuerza, hipertrofia, recomposición)
